  • 报错信息是关于 sqlite3 编译失败的。经过搜索，是因为找不到 /bin/sh 下的 python 命令，可惜 macOS 下没有 /bin/sh 目录，当前也只有 python2.x 和 python3.x 的命令，添加 python 环境变量也没效果。以下命令有效
npm install --build-from-source --python=/usr/bin/python3 sqlite3
  • 但这也不是解决问题的根本办法，又查到 sqlite3 不支持过高的 node 版本，切换到 node lts/fermium 14.x 尝试。安装成功。
