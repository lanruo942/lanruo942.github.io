影视字幕的制作与导入

2021-12-25 · 3 min read
Subtitle

下载影视资源的时候，经常需要单独寻找字幕。通常都能找到合适的字幕文件，不过也还是会有遇到问题的时候。比如：

  1. 发行版与导演剪辑版长度不同，某些片段字幕对不上。
  2. 有些字幕对应的不是原版视频，而是裁剪后的视频，所以整体时间错位。
  3. 分享给他人的时候，最好将外挂字幕压入视频。（字幕分为软字幕和硬字幕，软字幕分为外挂字幕和内嵌字幕）

目前常见的字幕文件格式有 srt、ass、sup，第三种比较少见，先不做过多了解。

SRT

SRT 字幕通常可以直接使用文本编辑器进行编辑，格式简单，无法添加个性化样式。
Final Cut Pro 和 DaVinci Resolve 都可以直接导入/导出，进行编辑和修改。

DaVinci Resolve 打开后切换到 Edit 页面，导入所需的视频和字幕文件，并拖动到时间轴。调整好字幕后导出即可。需要注意的是，默认时间轴的起始时间是 01:00:00:00，需要改为 00:00:00:00：Timeline 右键 -> Timelines -> Starting Timecode...

ASS

ASS 字幕可以添加丰富的个性化特效，但常规的视频剪辑软件（FCP、DaVinci、Premium）都不支持导入或编辑。经过搜索，可以制作和编辑 ASS 文件的软件只有 ArcTime 和 Aegisub。
ArcTime 的制作商应该是中国公司，风评差，软件中各种需要登录、付费的操作。软件风格个人也比较抵触。
Aegisub 是许多字幕组都在使用的，只不过最后一个 Release 版本在 2014 年。好在当前在 M1 Mac 上还可以正常运行。

Aegisub 使用：

  • 导入字幕文件
  • 导入视频/音频
  • 调整/编辑后导出

Aegisub 在导入中文字幕的时候出现乱码的问题，打开 字幕（菜单） -> 样式管理器 -> Default -> 编辑，修改字体为其他中文字体，修改编码为中文。

内嵌字幕

可以将制作好的字幕文件使用 Permute 进行压入。
另外尝试了 Handbrake，不知道为什么即便没勾选 Burned In，依然需要重新渲染整个视频文件。
